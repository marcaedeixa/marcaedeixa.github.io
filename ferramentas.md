# Ferramentas

## Seleção
A ferramenta de seleção <img src="/assets/selecao.png" width="18"> está localizada na barra esquerda do projeto. Ela serve para selecionar elementos no canvas como formas, textos, objetos e atores. Para utilizá-la, deve-se usar o botão esquerdo do mouse.

## Zoom
Pode-se alterar o zoom do canvas de duas formas diferentes:

**Opção 1 -** Selecionando os ícones de lupa <img src="/assets/zoomout.png" width="18"> <img src="/assets/zoomin.png" width="18"> da barra superior do projeto é possível aumentar e diminuir o zoom. Selecionar a seta <img src="/assets/zoom100.png" width="18"> entre os dois ícones de zoom faz com que ele volta à visualização de zoom 100%

**Opção 2 -** Pressionando a tecla `Ctrl` e usando a roda de rolagem é possível aumentar e diminuir o zoom

## Salvar
Todo projeto é salvo de forma automática periodicamente. No entanto, é possível salvá-lo manualmente selecionando o ícone de salvamento <img src="/assets/salvar.png" width="18">

## Exportar

Para todos os formatos de exportação deve-se primeiro selecionar o ícone de exportação <img src="/assets/exportar.png" width="18"> na barra superior do projeto.

### Projeto `.json` completo
Formato ideal para enviar o projeto para outro usuário da plataforma sem incluí-lo como colaborador. Para exportar, selecione **Exportar JSON**.

### Projeto `.svg` completo
Formato ideal para obter uma visão do projeto na forma de animação suportada por navegadores web. Para exportar, selecione **Exportar SVG**.

### Projeto `.png` completo
Formato ideal para salvar todas as cenas em arquivos de imagem. Para exportar, selecione **Projeto completo**. As imagens serão baixadas dentro de uma pasta compactada `.zip`.

### Cena `.png` individual
Formato ideal para salvar uma cena em arquivo de imagem. Para exportar, selecione a cena desejada.
