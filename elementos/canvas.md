# Canvas
{: .no_toc }

## Sumário
{: .no_toc .text-delta }

1. TOC
{:toc}
O canvas é o grid infinito no qual todos os elementos do projeto são incluídos.

## Mover a visualização no canvas
Para mover basta clicar com o botão direito do mouse em qualquer parte vazia do grid e arrastá-lo.

## Rolar verticalmente
Para rolagem vertical deve-se usar a roda de rolagem do mouse.

## Rolar horizontalmente
Para rolagem horizontal deve-se pressionar a tecla `Shift` enquanto usa a roda de rolagem do mouse.
