## Criar projeto
Para criar um novo projeto:
1. Acesse o Dashboard
2. Selecione **Criar Novo Projeto**
3. Especifique o nome do projeto
4. Selecione **Criar Projeto**

## Excluir projeto
Para excluir um projeto existente:
1. Acesse o Dashboard
2. Selecione **Excluir** no projeto que você deseja excluir
3. Selecione **Confirmar Exclusão**

obs.: Esta ação não pode ser desfeita.

## Compartilhar o projeto
Atualmente não é possível múltiplos usuários editarem o mesmo projeto, mas é possível fazer o download do projeto completo e encaminhar para que um outro usuário que tem conta na plataforma edite-o.
1. Acesse o Dashboard
2. Entre no projeto desejado
3. Selecione o ícone de exportação <img src="/assets/exportar.png" width="18">
4. Selecione **Exportar JSON**

## Renomear o projeto
Para renomear um projeto existente:
1. Acesse o Dashboard
2. Passe o mouse sobre o nome do projeto
3. Selecione o ícone de edição <img src="/assets/editar.png" width="18">
4. Altere o nome
5. Selecione o ícone de confirmação <img src="/assets/confirmacao.png" width="18">
