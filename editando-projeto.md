## Criar projeto
Para criar um novo projeto:
1. Faça login na plataforma
2. Acesse o Dashboard
3. Selecione **Criar Novo Projeto**
4. Especifique o nome do projeto
5. Selecione **Criar Projeto**

## Excluir projeto
Para excluir um projeto existente:
1. Faça login na plataforma
2. Acesse o Dashboard
3. Selecione **Excluir** no projeto que você deseja excluir
4. Selecione **Confirmar Exclusão**
obs.: Esta ação não pode ser desfeita.

## Renomear o projeto
Para renomear um projeto existente:
1. Faça login na plataforma
2. Acesse o Dashboard
3. Passe o mouse sobre o nome do projeto
4. Selecione o botão de edição
5. Altere o nome
6. Selecione o botão de confirmação
